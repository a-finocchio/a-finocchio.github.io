3D visual SLAM and Path Planning with drone

- Implemented LSD-SLAM in ROS Kinetic using a drone with mono-camera to generate the 3D point cloud map of its surrounding environment and localize itself in real-time.
- Converted the point cloud data to octo-map and utilized move group interface in MoveIt! to plan a collision-free path to goal position.
- Executed the collision-free path using the drone and simulated its movement in Gazebo.

Auto pick-and-place and image drawing using UR5 robot arm

- Implemented feedback control algorithms to drive the UR5 to move utilizing ROS Kinetic and Matlab.
- Calibrated its hand-eye and planed collision-free paths with Rapidly-exploring Random Tree algorithm.
- Programed the UR5 to draw contours of given images utilizing computer vision algorithms.

Simulation of planar mobile robot's motion planning

- Generated configuration space and plan collision-free movement using potential field algorithm.
- Implemented Extended Kalman Filter algorithm to track a three-wheel robot and simulated in Matlab.

Signal processing using machine learning algorithms

- Detected human faces in images and classified their genders using Adaboost classification algorithm.
- Recognized handwriting digits using Hidden Markov Model (HMM) method and Gaussian classifier.
- Identified different speakers from sound data using Gaussian Mixture Models (GMM) algorithms.

Detection of surface macro-features using tactile sensor

- Collected force signals by sliding tactile sensor on the plates with different texture features.
- Processed signal data and applied machine learning algorithms to classify the signals and their textures.
